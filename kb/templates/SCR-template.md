# Template para Scripts (SCR)

Los scripts se guardan como archivos ejecutables con un header de metadata en comentarios.
A continuación se muestra el formato para cada lenguaje.

---

## Bash (.sh)

```bash
#!/bin/bash
# ---
# id: "SCR-YYYY-####"
# titulo: ""
# servicio: ""
# lenguaje: "bash"
# uso: ""
# riesgos: ""
# dependencias: []
# relacionados: []
# tags: []
# fecha_creacion: "YYYY-MM-DD"
# autor: ""
# estado: "borrador"
# historial_cambios:
#   - fecha: "YYYY-MM-DD"
#     autor: ""
#     descripcion: "Creación inicial"
# ---

# --- Script body ---
```
